# Prompt Engineering
### Prompt Structure
1. Role
2. Task
3. Directions
    - Specifics
    - Steps
    - Dos and Don'ts
4. Output format
5. Examples
    - Providing good Examples creates a good response also so give as much examples as you could for the LLM
6. Input Data
